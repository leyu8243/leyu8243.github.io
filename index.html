Weather Station Project

By Leyu Girma


Project Overview

In this course, I have created a weather station which reads the weather in terms of temperature, humidity, barometric pressure, and UV radiation. To collect this data, I used a Raspberry Pi pico board as well as a collection of Adafruit sensors which were connected with Arduino.


What I've Learned

Over the course of this week, I have learned how to use electronics and circuit boards, connect microcontrollers to computer programs, train AI, and finally create a website such as this one utilizing HTML.
